=head1 NAME

Data::DublinCore::Util - DublinCore XML useful constants

=head1 INHERITANCE

 Data::DublinCore::Util
   is a Exporter

=head1 SYNOPSIS

  use Data::DublinCore::Util qw/:dc20021212/;

=head1 DESCRIPTION

Defines the namespaces used with DublinCore XML schemas.

=head1 FUNCTIONS

=head2 Constants

Defined are C<NS_DC_DCMITYPE>, C<NS_DC_ELEMS11>, C<NS_DC_TERMS>.
You may also need C<XMLNS> or C<SHEMA2001> from L<XML::Compile::Util|XML::Compile::Util>.
